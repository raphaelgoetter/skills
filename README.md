# skills — Instructions pour IA (Claude/Copilot)

Ce dossier contient les guidelines de référence pour les assistants IA en charge du front-end.

## Objectif

- Centraliser les règles de style et de comportement pour les générations de code.
- Fournir un guide rapide pour les contributeurs et agents sur la stack Alsacréations.

## Contenu

- `CLAUDE.md` : règles prioritaires si Claude est utilisé.
- `AGENTS.md` : fallback générique pour tout agent IA.

## Utilisation

1. Consulter `CLAUDE.md` avant d’écrire du HTML/CSS/JS.
2. Vérifier l'usage des layouts Bretzel et tokens CSS.
3. Signaler toute demande conflictuelle avec ces directives.
